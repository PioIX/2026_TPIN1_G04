### Descripción de la propuesta
El juego va a ser inspirado en el “Rosco”, pasapalabra, de temática variada. Se va a
mostrar por pantalla el Rosco e irán apareciendo preguntas o sentencias para que el usuario
responda según corresponda. Debajo aparecerán dos botones, uno para enviar la respuesta y
otro para saltear la consigna (“Pasapalabra”). Si el usuario responde incorrectamente, la
letra de la consigna se pondrá de color rojo; si elige la opción “Pasapalabra”, se pondrá de
color amarillo y pasará a la siguiente letra; por último, si responde correctamente, la letra
se volverá de color verde.
